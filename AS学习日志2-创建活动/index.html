创建活动

1. 关掉当前的project 新建一个project

点击file->close project 关掉当前project

新建project 选择 add no activity 点击next ，Name键入 Activity Test Language 选择 java， finish

2. 手动创建活动

项目创建成功后，默认使用Android模式的项目结构，我们手动改成Project模式

可以看到 app/src/main/java/com.example.activitytest目录为空

右击com.example.activity->new->activity->Empty Activity,将活动命名为FirstActivity，不要勾选Generate Layout File 和Launcher Activity，勾选Generate Layout File 表示自动为FirstActivity创建一个对应的布局文件，勾选Launcher Activity表示自动将FirstActivity设置为当前项目的主活动，

创建和加载布局

Android程序设计讲究逻辑和视图分离，最好每个活动对应每个布局。

右击app/src/main/res 右击后->new->Directory,先创建一个名为layout 的目录。右击layout->new->layout resource file, 弹出一个新建布局资源文件的窗口，将这个布局文件命名为first_layout,根元素默认选择LinearLayout 如图

完成创建会看到布局编辑器

切换到text看到代码，点击右边的preview预览布局，

向布局中新建一个button在first_layout.xml中

button属性：

  • android:id是当前元素的唯一表示符，之后代码会用这个id对元素进行操作。
  • android：layout_width是元素宽度
  • android：layout_width是元素长度，match_parent表示与父属性相同，wrap_parent表示元素高度刚好包含里面的内容。

Toast是Android提供的一种提醒方式，可以使用它将一些短小的消息通知给用户，一段时间后自动消失。

findViewById()获取到布局文件中定义的元素，来得到按钮的实例。返回View对象， 向下转型成Button对象。

得到实例后，调用setOnClickListener()方法为按钮注册一个监听器，点击按钮会执行监听器中的onClick()方法，所以弹出Toast的功能要在onClick中编写。

通过makeText()创建出一个Toast对象，调用show()显示。nakeText()三个参数，第一个Context，活动本身就是一个Context对象，因此直接传入FirstActivity.this 第二个Toast显示的文本内容，第三个Toast时长。
